
## Интерактивная карта для отображения плана этажа со станками для завода

### Стэк:
- Kotlin Multiplatform
- Compose Multiplatform
- JSON
- androidx-lifecycle, viewmodel, material3, kotlinx-coroutines, kotlinx-serialization

### Функции:

- отображение информации о станке, при наведении курсора на станок
- контроль и отображение даты планового техобслуживания
- отображение дополнительной информации и комментариев о станке, при нажатии на станок
- управление пользователями
  - добавление
  - удаление
  - уровни доступа пользователей по ролям
- шифрование паролей пользователей и администратора
- редактирование информации о станке согласно роли пользователя
  - добавление комментариев
  - редактирование даты техобслживания
